Details

Fontmarklet is currently in Alpha. Do not expect anything to work.

Fontmarklet is a bookmarklet made for designers and web developers who want to try out fonts on websites without having to dive into the source every time.

If you want to contribute or report bugs, check out the GitHub repository.

Check it out on GitHub Tweet it! Plus One!

Features

All the fonts you need
Try out all of Google WebFonts or any font on your local machine
Just a bookmark
Small and lightweight, super easy to install and remove. Does not drag your resources. You decide when you want it to be running.
Works (almost) everywhere
Use it with any browser or OS! Mobile support in the works.
Keeps your ideas
Automatically saves your font configurations on every website.
